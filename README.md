**Chatbot-Based Music Recommendation System**

A smart and interactive Music Recommendation System powered by Machine Learning and NLP (Natural Language Processing) that suggests songs based on user mood through a chatbot interface.

This project enhances user experience by combining AI-driven conversation with personalized music recommendations.

**Features**
---Chatbot Interaction
Conversational chatbot interface
Understands user mood using NLP
Provides real-time responses
---Music Recommendation
Suggests songs based on:
Mood (Happy, Sad, Romantic, etc.)
User preferences
Personalized recommendations
---User Management
User Registration & Login
Password recovery system
Profile-based recommendations
---Machine Learning
Mood detection using NLP
Sentiment analysis from chat
Continuous improvement via feedback
----Music Features
Play recommended songs
Song selection based on mood
Admin can upload songs
